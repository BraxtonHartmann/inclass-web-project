Why Computer Science?

For as long as I can remember I have always been intrigued and fascinated by riddles, puzzles, mazes and locks. I've almost had an obsession over these things and understanding the inner workings of machines and utilities when they broke down. as I've grown up I have found more and more appreciation for game and puzzle design. and at first when I started taking CS classes I was really excited about the ability for me to have tools and problem solve by using those tools. I really enjoyed programming because it is a lot like a large puzzle that I can work out and reverse engineer so that I can have it achieve what I would like it to. I've become even more excited since I've started taking this class because of one of those hidden secret style of puzzle called an ARG (Alternate Reality Game). the reason for this is because oftentimes these ARGs take place across social media, websites with hidden elements, YouTube videos and many other sources. but because I have been currently working on one of these projects myself I look forward to being able to apply what I'm learning with this class directly to one of my hobbies.
